Here's a list of supported languages that will be transformed by [highlight.js](https://highlightjs.org/)


## JavaScript

```js
var pet = {
    name: "Frida",
    age: 13
}
```

## HTML

```js
<!doctype html>
<html lang="en">
    <head>
        <title>doc GOV</title>
    </head>
    <body class="page ">
    </body>
</html>
```

## CSS

```css
@media (max-width: 850px) {
  /* only show nav toggles on small screens */
  .hamburger {
    display:flex
  }
}
```
